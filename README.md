# Visitor Management System (VMS)

## Project Overview
This system manages visitors from appointment booking to exit using QR codes across Gate 1, Reception, and Gate 2.

## DevOps Contribution (Louisa – DevOps Engineer)

As the DevOps Engineer, my role focuses on:
- Environment setup and configuration
- System workflow documentation
- Deployment planning
- Supporting integration between components

## Setup Instructions

1. Install Python (3.x)
2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py

## Key Features
- Appointment scheduling system
- QR code-based check-in
- Visitor tracking across entry and exit points
- Analytics dashboard
- Data export (CSV/PDF)

## DevOps Notes
- Use `.env` file for environment variables
- Ensure system supports offline data upload (CSV)
- Plan for future deployment (local server or cloud)
